What This Tool Does

Do you often end up with hundreds of RAW and JPG photos in the same folder? This utility helps you quickly organize them by:

  • Finding all RAW files that have a matching JPG with the same name
  • Moving both the RAW and the JPG into a separate folder called matched_raws
  • Saving you hours of manual sorting!

How to Use It

  1. Download the Tool using the form above.
  2. Unzip the File to get filter_raws.bat.
  3. Place It in Your Photo Folder (the one with RAW + JPG files).
  4. Run the Tool by double-clicking the filter_raws.bat file.
  5. A new folder called matched_raws will contain all the matched pairs.

Example

If your folder has:

IMG_001.CR2
IMG_001.JPG
IMG_002.CR2
IMG_003.JPG

After running the tool, matched_raws will contain:

IMG_001.CR2
IMG_001.JPG

The rest stay in the original folder.
